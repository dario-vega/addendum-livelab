## Using hierarchy 

### Parent table


```
CREATE TABLE IF NOT EXISTS stream_acct_hierarchy (
  id INTEGER GENERATED BY DEFAULT AS IDENTITY
    (START WITH 4 INCREMENT BY 1 NO CYCLE),
  country   STRING,
  firstName STRING,
  lastName  STRING,
  PRIMARY KEY(id)
)
```

### Child table: use showId as the key 

```
CREATE TABLE IF NOT EXISTS stream_acct_hierarchy.shows (
  showId INTEGER,
  showName STRING,
  type STRING,
  numSeasons INTEGER,
  genres ARRAY(STRING),
  seriesInfo JSON,
  PRIMARY KEY(showId)
)
```

### QUERIES 

```
SELECT * 
FROM NESTED TABLES 
(stream_acct_hierarchy a descendants(stream_acct_hierarchy.shows b))
```

#### Previous Q1
```
select $show.showId, count(*) as cnt
from stream_acct u, unnest(u.info.shows[] as $show)
group by $show.showId
order by count(*) desc
```
#### New Q1
```
SELECT $show.showId, count(*) as cnt 
FROM NESTED TABLES 
(stream_acct_hierarchy u descendants(stream_acct_hierarchy.shows $show))

order by count(*) desc
```

#### Previous Q2

```
select $show.showId, sum($show.seriesInfo.episodes.minWatched) as totalTime
```
